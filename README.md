# 基于Vue3开发的我的空间

## 1. 项目介绍

本项目开发了一个简易版的 QQ 空间，用于学习 Vue3 开发框架。

## 2. Project setup

```
npm install
```

### 2.1 Compiles and hot-reloads for development

```
npm run serve
```

### 2.2 Compiles and minifies for production

```
npm run build
```

### 2.3 Lints and fixes files

```
npm run lint
```
